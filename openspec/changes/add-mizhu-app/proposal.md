# 迷住团购管家

## 概述

创建一个纯本地 PWA 应用，用于记录和统计"迷住"团购活动中的所有下单和返现情况。应用帮助团购参与者解决订单多且零碎、手动计算返现易错、团购周期长容易忘记待收款等痛点。

## 动机

迷住团购的返现机制极其复杂，包含 7 种返现类型、2 种保价类型、以及活动级别的大套购分档计算。参与者目前依赖手动 Excel 记录，存在以下痛点：

- **计算易错**：品牌套购、大套购、保价等计算有层层依赖，手动算容易出错
- **忘记待收款**：返现周期长（确认收货后 14 天内到账，大套购手动结算），订单多时容易忘记哪些钱没收
- **状态追踪困难**：店铺侧和迷住侧两路返现分别到账，追踪混乱

## 方案

构建一个纯本地 PWA（Progressive Web App），数据全部存储在设备本地（IndexedDB），不依赖任何云服务。Android 和 iOS 用户均可通过浏览器安装使用，无需上架应用商店。

### 核心能力

1. **活动管理**：创建/编辑团购活动，配置大套购规则
2. **商品录入**：单页折叠式录入，包含基本信息、返现明细、品牌套购
3. **返现计算引擎**：自动计算品牌套购基准、大套购分档分摊、保价差价
4. **状态追踪**：商品状态流转（待收货→待返现→已完成/已取消），每笔返现独立追踪
5. **待返清单**：按到账时间线分组展示，支持超期提醒
6. **大套购结算**：支持分批结算和全部结算，支持整批撤销
7. **批量确认收货**：批量触发返现 14 天倒计时
8. **统计报表**：活动总览、返现构成、历史趋势
9. **数据导出**：JSON（完整结构）和 Excel（扁平化表格）

### 技术栈

- Vue 3 + TypeScript + Vite
- Vant（移动端 UI 组件库）+ Tailwind CSS
- Dexie.js（IndexedDB 封装）
- vite-plugin-pwa（离线支持 + 桌面安装）
- ECharts（统计图表）

## 范围

### V1 包含

- 全部核心业务逻辑（7 种返现类型、2 种保价、大套购分批结算）
- 全部 9 个核心页面
- 数据导出（JSON + Excel）
- 纯本地存储，无后端

### V1 不包含

- 数据导入/恢复
- 多设备数据同步
- 自动备份提醒
- iOS App Store 上架
- 推送通知

## 风险

- **数据丢失风险**：IndexedDB 存于浏览器沙箱，清除浏览器数据会丢失全部数据。V1 不做备份提醒，用户需自行导出 JSON 备份。
- **多设备数据隔离**：不同设备/浏览器的数据不互通，V1 不额外说明。
- **浮点数精度**：JavaScript 浮点数计算存在精度问题，计算引擎统一使用整数分（×100）运算。
