# 活动管理

## 需求

创建、编辑团购活动，配置大套购规则。

## 规格说明

### 活动字段

- 活动名称（用户自定义，必填）
- 开始日期（选填）
- 结束日期（选填）
- 大套购规则（档位列表，默认带出）
- 活动状态（进行中 / 已完结）

### 创建活动

- 空状态首页点击"新建活动"进入创建表单
- 大套购规则默认带出全局配置：[3000→100, 5000→200, 10000→500]
- 支持添加/删除/修改档位
- 创建后活动出现在首页下拉选择列表中

### 编辑活动

- 通过活动下拉列表中的"编辑"入口进入
- 表单复用创建页，标题改为"编辑活动"，字段预填已有数据
- 修改活动信息和大套购规则
- 已结算的批次不受大套购规则修改影响

### 活动状态

- 进行中：默认状态
- 已完结：用户手动标记，不影响数据计算

### 活动切换

- 首页顶部居中显示当前活动名称，点击展开下拉列表
- 下拉列表展示所有活动（含状态标签）
- 下拉列表中每个活动右侧有"编辑"入口，点击进入编辑表单
- 下拉列表底部有"新建活动"入口
- 所有数据页面（首页/商品清单/待返清单/统计）均基于当前选中活动
