# 商品录入

## 需求

用户以单个商品为单位录入订单信息，支持快速录入和后续补充。

## 规格说明

### 录入表单（单页折叠式）

**基本信息（默认展开）：**
- 型号/商品名（必填，文本）
- 下单途径（必填，下拉选择：淘宝/京东/小程序/其他）
- 订单号（选填，文本，多个商品可共享同一订单号，不作为分组标签）
- 下单价（必填，数字，保留2位小数）
- 赠品（选填，文本）

**返现明细（折叠，可跳过）：**

店铺侧：
- 下单即返（勾选 + 金额输入）
- 小红书晒单（勾选 + 金额输入）
- 组合购（勾选 + 金额输入）
- 其他（勾选 → 自定义条目名 + 金额输入，支持多个）

迷住侧：
- 迷住返(加返)（金额输入）
- 舰长神券（金额输入）
- 装修神券（金额输入）
- 舰长神券与装修神券互斥（选一个自动取消另一个），可以都不选

**品牌套购（折叠，可跳过）：**
- 选项1（系数输入）
- 选项2（系数输入）
- 自定义（系数输入）
- 不选 = 不参与品牌套购
- 用户填入系数后立即生成 CashbackEntry(type='brand', isEstimated=false)
- 基准依赖其他 Entry，每次 Item 更新后自动重算

### 保存校验

- 必填字段缺失 → 阻止保存，提示具体缺哪个
- 返现总额 > 实付金额 → 阻止保存，提示"返现总额不能超过下单价"
- 到手价 < 0 → 阻止保存

### 保存后

- 商品状态默认为"待收货"
- 自动生成 CashbackEntry(type='big_purchase', amount=price×0.05, isEstimated=true)（仅活动启用大套购时）
- 如填了品牌套购系数，生成 CashbackEntry(type='brand')
- 触发动态重算（品牌套购/迷住保价）
- 跳转到商品详情页或返回清单页

### 编辑限制

- 录入时不出现保价字段，编辑时才显示
- 已参与大套购结算的商品，下单价锁定不允许修改
