# 数据导出

## 需求

支持将活动数据导出为 JSON 和 Excel 两种格式。

## 规格说明

### JSON 导出

- 导出当前活动的完整数据结构
- 包含：活动信息、所有商品、所有返现条目、所有收款记录、所有结算批次
- 用于备份（V1不支持导入恢复）

### Excel 导出

- 扁平化表格，每件商品一行
- 列设计参考用户现有 Excel 表格结构
- 自定义返现条目（"其他"类型）合并为"其他"一列，金额求和
- 标准条目保持独立列（下单即返、小红书晒单、组合购、品牌套购等）

### 导出范围

- 导出当前选中活动的数据
- 支持导出全部活动（JSON 格式）

### 触发方式

- 统计报表页面或设置中的导出入口
- 导出后通过浏览器下载保存文件
