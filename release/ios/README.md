# 🍎 iOS 安装指南

## 系统要求

- iOS 15.0 或更高版本
- Safari 浏览器（PWA 仅支持通过 Safari 安装）

## 安装步骤

1. 打开 **Safari** 浏览器（⚠️ 必须是 Safari，Chrome/微信内置浏览器无法安装）

2. 在地址栏输入应用网址并访问

3. 点击底部工具栏的 **「分享」按钮**（方框中向上箭头的图标）

4. 在弹出菜单中向下滑动，找到并点击 **「添加到主屏幕」**

5. 确认标题（可自定义名称），点击右上角 **「添加」**

6. 返回主屏幕，可以看到应用图标，点击即可全屏启动

## 使用说明

- 从主屏幕启动时，应用以全屏模式运行，没有浏览器地址栏
- 支持离线使用（首次加载后，后续打开无需网络）
- 数据存储在本地设备中，卸载应用会清除数据
- 如需清除数据：设置 → Safari → 高级 → 网站数据 → 删除对应站点

## 常见问题

**Q: 为什么微信里打不开？**
A: 微信内置浏览器不支持 PWA 安装。请点击右上角「···」→「在浏览器中打开」，然后使用 Safari 安装。

**Q: 更新后没有变化？**
A: 应用会在后台自动更新。如果没生效，请长按桌面图标删除后重新安装（数据不会丢失）。

**Q: 存储空间占用？**
A: 约 5-10 MB（含缓存数据）。
