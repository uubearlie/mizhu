## V1.0.0 (2026-06-26)

🎉 首次正式发布！

### 核心功能
- **活动管理**：创建/编辑/切换团购活动，配置大套购阶梯返现规则（默认：满3000返100、满5000返200、满10000返500）
- **订单管理**：录入商品信息、下单价、下单途径、赠品等
- **返现明细**：三种来源独立追踪
  - 店铺返现：下单即返、小红书晒单、组合购、自定义条目、品牌套购、平台/店铺保价退款
  - 迷住返现：迷住加返、舰长神券、装修神券、迷住保价
  - 大套购：预估5%自动生成，支持手动结算与撤销
- **保价管理**：平台/店铺保价退款记录，确认收货时平台保价（即时到账）自动创建收款记录
- **收到返现**：弹窗内点击待返金额可快速填充，支持超额校验
- **大套购结算**：前置规则（店铺+迷住返现全部到账才可结算），批量结算 + 撤销
- **数据统计**：返现构成图表、累计统计、JSON/CSV 导出
- **待返提醒**：首页到期提醒（逾期/即将到期）

### 交互优化
- 待返页面 Tab 页签（订单返现 / 大套购），大套购按可结算/不可结算分组显示
- 编辑页面支持修改返现条目金额和新增自定义条目
- 待收货商品卡片直接显示确认收货按钮
- 删除大套购返现记录时弹出提醒，连带撤销同批次
- 全屏使用说明（计算逻辑 + 操作流程）

### UI 设计
- 小清新风格主题（柔和蓝 #5b8def）
- Vant ConfigProvider 统一主题色
- 卡片圆角 16px + 柔和阴影 + 细边框
- 首页概览卡片渐变背景

### 技术架构
- Vue 3 + TypeScript + Vant 4 + Vite
- Dexie.js (IndexedDB) 本地存储，支持版本数据迁移
- PWA (Workbox) 离线缓存 + 自动更新
- 数据迁移机制：DB schema v4，含大套购 payer 自动迁移脚本
